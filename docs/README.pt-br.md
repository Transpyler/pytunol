# Bem vindos ao Pytunhol!


Pytunhol é uma extensão do Python que aceita comandos em espanhol (assim como os comandos tradicionais em inglês). Ela foi criada utilizando as ferramentas do projeto Transpyler como uma demonstração de como criar versões traduzidas para o Python.

Pequena curiosidade: eu não falo espanhol! O Pytuñol utiliza o google translate para realizar traduções automáticas. O projeto transpyler foi originalmente criado para suportar o Pytuguês e depois foi reescrito em inglês para funcionar com qualquer idioma. Minha vontade original era traduzir do português para o espanhol (daí a alusão ao portunhol), mas acabou sendo mais prático basear todas as traduções na versão em inglês do Transpyler. 

Este projeto existe como um modelo para quem quiser iniciar uma nova tradução do Python e para provocar a comunidade falante de espanhol a assumir a ferramenta e criar uma linguagem séria a partir daí. (Alguém pode se voluntariar e fazer um cookie cutter?)
