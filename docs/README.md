# ¡Bienvenidos al Pytunhol!


Pytunhol es una extensión de Python que acepta comandos en español (así como los comandos tradicionales en inglés). Se creó utilizando las herramientas del proyecto Transpyler como una demostración de cómo crear versiones traducidas para Python.

Pequeña curiosidad: ¡no hablo español! Pytuñol utiliza el google translate para realizar traducciones automáticas. El proyecto transpyler fue originalmente creado para soportar a Pytgués y luego fue reescrito en inglés para funcionar con cualquier idioma. Mi voluntad original era traducir del portugués al español (de ahí la alusión al portunhol), pero acabó siendo más práctico basar todas las traducciones en la versión en inglés del Transpyler.

Este proyecto existe como un modelo para quien quiera iniciar una nueva traducción de Python y para provocar a la comunidad hablante de español a asumir la herramienta y crear un lenguaje serio a partir de ahí. (Alguien puede voluntariarse y hacer un cookie cutter?)
